구인구직 플랫폼 기획안

1. 서비스 개요

theBio 는 기업 회원과 개인 회원을 대상으로 채용 공고 게시, 이력서 등록, 지원현황 관리 기능을 제공하는 초간단 구조의 실용적 구직 플랫폼입니다.


목표는 심플하고 직관적인 UI/UX로 빠르고 효율적인 구인/구직 경험을 제공합니다.

2. 주요 사용자 유형

  • 개인회원: 이력서 업로드, 채용공고 열람, 지원현황 확인
  • 기업회원: 공고 작성, 상품 구매, 공고 관리 및 끌어올림 기능 제공

3. 기존 채용사이트와의 차별점

  • 저렴한 비용으로 실속 있게 시작 가능
  • 헬스케어·바이오 산업군에 특화된 직군/기업 중심 채용 공고 운영
  • 개인회원은 초간단 PDF 이력서 업로드만으로 지원 가능 (회원가입 후 즉시 지원)
  • 기업회원은 전달된 PDF 이력서 미리보기 지원, 별도 다운로드 없이 확인
  • 복잡한 절차 없이 빠른 채용·지원 흐름에 집중

5. 핵심 기능 흐름

[개인회원]

  • 프로필 관리: 기본 정보 + 휴대폰 인증
  • 이력서 관리: PDF 업로드 및 리스트
  • 지원현황: 공고명, 상태, 지원일자 확인

[기업회원]

  • 공고 등록: 상품 선택 → 결제 → 게시글 작성
  • 공고 관리: 수정 / 마감 / 끌어올림
  • 결제 수단: 토스페이먼츠 카드결제 연동

6. 기술 스택 요약

  • 프론트엔드: Next.js, TailwindCSS
  • 인증: 휴대폰 인증 (NAVER SENS / CoolSMS 등)
  • 파일: PDF 업로드 및 react-pdf 뷰어
  • 배포: Vercel / Naver Cloud 등
  • 결제: Toss Payments 연동
